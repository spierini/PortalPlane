Santi Pierini
Final Program
“Portal Plane”

CONTROLS:

-Double finger mousepad scrolling required for Pitch & Yaw controls

-Press UP-Arrow to start motion forward

-Press SPACE to reset position after crash or anytime

-Press A or D to strafe plane left and right



From inside the project directory, run the commands:

mkdir build
cd build
cmake ..
make -j4
./final ../resources
